Linear transform definitions and properties

A linear transformation is a function that takes a vector as input and returns a transformed vector as output. Linear transformations have the following properties:



  • They preserve the vector space structure of the input space. This means that if two vectors are added together or multiplied by a scalar, the resulting vector is also in the input space.
  • They preserve the vector length. This means that the length of the transformed vector is equal to the length of the original vector.
  • They preserve the vector direction. This means that the direction of the transformed vector is the same as the direction of the original vector.
